# Prompt 05: Review The Agent Loop

Copy this into your agent chat after the first examples pass:

```text
Please review whether this agent workflow is now safer and easier to resume.

Check:
1. Are important decisions written down?
2. Is long-term memory still concise?
3. Are task status changes sourced?
4. Can another agent resume from the checkpoint?
5. Does the checkpoint say what not to redo?
6. Are private files, secrets, logs, and raw messages protected?
7. Are we still keeping this manual instead of over-automating?

Return:
- what is working
- what is still weak
- what should wait
- what one small follow-up would help most
```
