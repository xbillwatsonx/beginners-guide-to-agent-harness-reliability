# Prompt 01: Audit My Current Memory And Task Setup

Copy this into your agent chat:

```text
Please inspect how you currently store memory, task status, project notes, reports, and handoffs.

Do not change files yet.

Report:
1. Where durable memory appears to live.
2. Where daily/session notes appear to live.
3. Where task tracking appears to live.
4. Where project docs or reports appear to live.
5. Whether any files look too large or too mixed together.
6. What should not be copied into memory, such as secrets, raw logs, credentials, caches, or private messages.
7. A small recommended reliability plan.

Keep the plan simple and ask before creating new files.
```
